## Setup
1. Clone this repo:
   ```bash
   git clone https://github.com/Syaekhul/notes-apps.git
   cd notes-apps
   ```
2. Install dependency:
   ```bash
   npm install
   ```
3. Run the app locally:
   ```bash
   npm run start-dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
